    \({\large \textbf{7.1 The Matrix Inverse}}\)
    \({\large \textbf{Definition}}\text{:}\)
    🔗
    The \(n \times n\) matrix \(A\) is \({\textbf{invertible}} \textup{ (or } \textbf{non-singular} \textup{)}\) if there is an \(n \times n\) matrix \(C\) so that
    🔗
    \begin{equation*} AC = CA = I_n\text{.} \end{equation*}
    🔗
    If there is, we write \(C = A^{-1}\text{.}\)
    🔗
    \(\)
    🔗
    \(\)
    🔗
    Note that
    🔗
    • A matrix that is not invertible is \(\textbf{singular}\text{.}\)
      🔗
      🔗
    • There is a formula for computing the inverse of a \(2 \times 2\) matrix.
      🔗
      🔗
    \({\large \textbf{Theorem}}\text{:}\)
    🔗
    The \(2 \times 2\) matrix \(\begin{pmatrix}a \amp b \\ c \amp d \end{pmatrix}\) is non-singular if and only if \(ad - bc \neq 0\text{,}\) and
    🔗
    \begin{equation*} {\begin{pmatrix} a \amp b \\ c \amp d \end{pmatrix}}^{-1} = {\frac 1 {ad-bc}} \begin{pmatrix}d \amp {-b} \\ {-c} \amp a \end{pmatrix} \end{equation*}
    \(\)
    🔗
    \(\)
    🔗
    \({\textbf{Example}}\text{:}\)
    🔗
    Let \(A = \begin{pmatrix} 1 \amp k \\ 2 \amp 1 \end{pmatrix}\text{.}\)
    🔗
    1. For what values of \(k\) is \(A\) singular?
      🔗
      🔗
    2. Use the inverse to solve the system of equations
      🔗
      🔗
    \begin{equation*} x_1 + x_2 = 3 \end{equation*}
    \begin{equation*} 2x_1 + x_2 = 1 \end{equation*}

    \({\textbf{The Invertible Matrix Theorem}}\text{:}\)
    🔗
    \({\large \textbf{Theorem}}\text{:}\)
    🔗
    If \(A\) be an \(n \times n\) matrix then these statements are equivalent.
    🔗
    1. \(A\) is invertible.
      🔗
      🔗
    2. \(A\) is row equivalent to \(I_n\text{.}\)
      🔗
      🔗
    3. \(A\) has \(n\) pivotal columns (all columns are pivotal).
      🔗
      🔗
    4. \(A\vec{x} = \vec{0}\) has only the trivial solution.
      🔗
      🔗
    5. The columns of \(A\) are linearly independent.
      🔗
      🔗
    6. The equation \(A\vec{x} = \vec{b}\) has a solution for all \(\vec{b} \in \mathbb{R}^n\text{.}\)
      🔗
      🔗
    7. The columns of \(A\) span \(\mathbb{R}^n\text{.}\)
      🔗
      🔗
    8. There is a \(n \times n\) matrix \(C\) so that \(CA=I_n \textup{ (} A \textup{ has a left inverse.)}\)
      🔗
      🔗
    9. There is a \(n \times n\) matrix \(D\) so that \(AD=I_n \textup{ (} A \textup{ has a right inverse.)}\)
      🔗
      🔗
    10. \(A^T\) is invertible.
      🔗
      🔗
    \(\)
    🔗
    \(\)
    🔗
    By equivalent statements we mean that:
    🔗
    • if one statement is true, all statements are true
      🔗
      🔗
    • if one statement is false, all statements are false
      🔗
      🔗
    \({\textbf{7.1.2 Elementary Matrices}}\text{:}\)
    🔗
    \({\large \textbf{Goal}}\text{:}\)
    🔗
    Represent row operations using matrix multiplication because it allows us to under- stand how algorithms that reduce matrices work.
    🔗
    \(\)
    🔗
    \(\)
    🔗
    Recall that we have elementary row operations:
    🔗
    • swap rows
      🔗
      🔗
    • multiply a row by a non-zero scalar
      🔗
      🔗
    • add a multiple of one row to another
      🔗
      🔗
    We can represent each operation by a matrix multiplication with an \(\textbf{elementary matrix}\text{.}\)
    🔗
    \({\large \textbf{Definition}}\text{:}\)
    🔗
    An elementary matrix, \(E\text{,}\) is a square matrix that differs by \(I_n\) by one row operation.
    🔗
    \({\textbf{Example}}\text{:}\)
    🔗
    Consider the sequence of row operations that reduce matrix \(A\) to the identity:
    🔗
    \begin{equation*} A = \underbrace{\begin{pmatrix} 1 \amp 2 \amp 0 \\ 0 \amp 0 \amp 2 \\ 0 \amp 1 \amp 0 \end{pmatrix}}_{A} \thicksim \underbrace{\begin{pmatrix} 1 \amp 2 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 2 \end{pmatrix}}_{E_1 A} \thicksim \underbrace{\begin{pmatrix} 1 \amp 2 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{pmatrix}}_{E_2 E_1 A} \thicksim \underbrace{\begin{pmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{pmatrix}}_{E_3 E_2 E_1 A} = I_3 \end{equation*}
    • Construct the elementary matrices \(E_1\text{,}\) \(E_2\text{,}\) and \(E_3\) that apply the row operations above.
      🔗
      🔗
    • Use the elementary matrices to construct \(A^{-1}\text{.}\)
      🔗
      🔗
