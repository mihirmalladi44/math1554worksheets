    \({\large \textbf{18.1 Orthogonal Projections}}\)
    🔗
    \(\textbf{Theorem}\)
    🔗
    • Suppose \(\vec{u}\) and \(\vec{y}\) are in \(\mathbb{R}\text{,}\) with \(\vec{u} \ne \vec{0}\text{,}\) and \(W = \text{Span} \{\vec{u}\}\text{.}\) The \(\textbf{projection of } \vec{y} \textbf{ onto } \vec{u}\) is the vector in the span of \(\vec{u}\) that is closest to \(\vec{y}\text{.}\)
      🔗
      🔗
    \begin{equation*} \text{projection of } \vec{y} \text{ onto } \vec{u} = \text{proj}_{\vec{u}} \vec{y} = \fillinmath{XXX} \vec{u} \end{equation*}
    • Moreover, \(\vec {y} = \hat {y} + \vec {z}\text{,}\) where \(\vec {z} \in W^\perp\)
      🔗
    \(\textbf {Geometric Interpretation}\)
    🔗
    The vector \(\vec {z} = \vec {y} - \text{proj}_{\vec {u}} \vec {y}\) is orthogonal to \(\vec {u}\text{,}\) so that
    🔗
    \begin{equation*} \vec{y} = \text{proj}_{\vec{u}} \vec{y} + \vec{z} \end{equation*}
    \begin{equation*} \|\vec{y}\|^2 = \|\text{proj}_{\vec{u}} \vec{y}\|^2 + \|\vec{z}\|^2 \end{equation*}

    \(\textbf {Theorem}\)
    🔗
    • An \(m \times n\) matrix \(U\) has orthonormal columns if and only if\(U^T U = I_n\text{.}\)
      🔗
      🔗
    \(\textbf {Definition}\)
    🔗
    • An \(\textbf {orthogonal matrix}\) is a square invertible matrix \(U\) so that \(U^T U = I\text{.}\)
      🔗
      🔗
    Note:
    🔗
    • In other words, an \(\textbf {orthogonal matrix}\) is a square matrix with \(\textbf {orthonormal columns}\text{.}\)
      🔗
      🔗
    • There is no name for matrices that have columns that are orthogonal but orthonormal

    \(\textbf {18.2.1 Properties of Matrices with Orthonormal Columns}\)
    🔗
    1. \(\displaystyle \|\vec{Ux}\| = \|\vec{x}\|\)
      🔗
    2. \(\displaystyle (\vec{Ux}) \cdot (\vec{Uy}) = \vec{x} \cdot \vec{y}\)
      🔗
    3. \((\vec{Ux}) \cdot (\vec{Uy}) = 0\) if and only if \(\vec{x} \cdot \vec{y} = 0\)
      🔗
    These properties tell us that the mapping \(\vec {x} \to U \vec{x}\) preserves lngth and orthogonality.
    🔗
