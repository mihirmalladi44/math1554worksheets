    \({\large \textbf{5.2 Worksheet Exercises}}\)

    1.

    Let \(A\) be a \(3 \times 4\) matrix. What must \(c\) and \(d\) be if we define the linear transformation \(T : \mathbb{R}^c \to \mathbb{R}^d\) by \(T(\vec{x}) = A\vec{x}\text{?}\)
    🔗
    🔗

    2.

    Let \(T : \mathbb{R}^2 \to \mathbb{R}^2\) be a linear transformation such that
    🔗
    \begin{equation*} T {\begin{pmatrix}{\begin{bmatrix}x_1 \\ x_2 \end{bmatrix}} \end{pmatrix}} = x_1 {\begin{bmatrix} -1 \\ 3 \end{bmatrix}} + x_2 {\begin{bmatrix}4 \\ -1 \end{bmatrix}} \end{equation*}
    Construct a matrix \(A\) so that \(T(\vec{x}) = A\vec{x}\) for all vectors \(\vec{x}\text{.}\)
    🔗
    🔗

    3.

    Let \(T : \mathbb{R}^4 \to \mathbb{R}^3\) be a linear transformation such that
    🔗
    \begin{equation*} T {\begin{bmatrix}4 \\ 0 \\ 1 \\ 0 \end{bmatrix}} = T {\begin{bmatrix} 0 \\ 0 \\ 1 \\ 4 \end{bmatrix}} \neq \vec{0}\text{.} \end{equation*}
    🔗
    Identify a non-trivial solution \(\vec{x}\) to \(T\vec{x} = \vec{0}\text{.}\)
    🔗
    🔗

    4.

    Let \(T_A\) be the linear transformation with the matrix below. Match each choice of \(A\) on the the left with the geometric description of the action of \(T_A\) on the right.
    🔗
    \begin{equation*} \begin{matrix} {\begin{bmatrix} .5 \amp 0 \\ 0 \amp .5 \end{bmatrix}} \hskip1in \amp \hskip1in \textup{rotation by }90^{\circ} \\ {\begin{bmatrix} 0 \amp 0 \\ 0 \amp 1 \end{bmatrix}} \hskip1in \amp \hskip1in \textup{A shear} \\ {\begin{bmatrix} 0 \amp -1 \\ 1 \amp 0 \end{bmatrix}} \hskip1in \amp \hskip1in \textup{Projection onto }y\textup{ axis} \\ {\begin{bmatrix} 1 \amp 1 \\ 0 \amp 1 \end{bmatrix}} \hskip1in \amp \hskip1in \textup{dilation by }1/2 \end{matrix} \end{equation*}
    🔗

    5.

    Indicate whether the statements are true or false.
    🔗

    (a)

    If \(A\) is a \(3 \times 2\) matrix then the map \(x \mapsto Ax\) cannot be one-to-one.
    🔗
    🔗

    (b)

    If \(A\) is a \(2 \times 3\) matrix then the map \(x \mapsto Ax\) cannot be onto.
    🔗
    🔗

    (c)

    The linear transform \(T_A : \mathbb{R}^n \to \mathbb{R}^m\) is one-to-one if and only if the only solution to \(A\vec{x} = \vec{0}\) is the trivial solution.
    🔗
    🔗
    🔗

    6.

    Construct the standard matrix of the linear transformation \(T\text{.}\)
    🔗

    (a)

    \(T : \mathbb{R}^2 \to \mathbb{R}^4\text{,}\) where \(T \begin{pmatrix}\begin{bmatrix}1 \\ 0\end{bmatrix}\end{pmatrix} = \begin{bmatrix} 3 \\ 1 \\ 4 \\ 1\end{bmatrix}\) and \(T\begin{pmatrix}\begin{bmatrix}0 \\ 1 \end{bmatrix}\end{pmatrix} = \begin{bmatrix}1 \\ 6 \\ 1 \\ 8\end{bmatrix}\)
    🔗
    🔗

    (b)

    \(T\) is a vertical shear given by \(T(\vec{e}_2) = 2\vec{e}_2\) and \(T(\vec{e}_1) = \vec{e}_1-2\vec{e}_2\text{.}\)
    🔗
    🔗

    (c)

    A matrix \(A \in \mathbb{R}^{2 \times 2}\) such that \(T (\vec{x}) = A\vec{x}\text{.}\) \(T\) is a linear transformation that first reflects vectors across the line \(x_1 = x_2\text{,}\) then rotates them counterclockwise by \(\pi\) radians about the origin, then reflects them across the line \(x_2 = 0\text{.}\)
