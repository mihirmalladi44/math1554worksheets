    \({\large \textbf{1.2 Linear Systems}}\)
    A \({\textbf{linear system}}\) is a set of linear equations. An example of a linear system with two equations is
    ðŸ”—
    \begin{equation*} x + 3y = 4 \end{equation*}
    \begin{equation*} 5x + 6y = 5 \end{equation*}
    We might want to know:
    ðŸ”—
    • what values of the unknowns satisfy all equations in the system, if any?
      ðŸ”—
    • what procedure do we want to use to identify those values?
